## The Fix Needed

**File:** `rangkai-marketplace/components/orders/BitcoinPaymentFlow.tsx`  
**Line:** ~91 (in generatePaymentAddress function)

**Current code:**
```typescript
const response = await fetch(`/api/orders/${order.id}/bitcoin-payment`, {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    amount: order.total_amount,
    currency: order.total_currency
  })
})
```

**Needs to be:**
```typescript
// Get token from auth context
const { token } = useAuth() // Add this at top of component

const response = await fetch(`/api/orders/${order.id}/bitcoin-payment`, {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json',
    'Authorization': `Bearer ${token}` // ADD THIS LINE
  },
  body: JSON.stringify({
    amount: order.total_amount,
    currency: order.total_currency
  })
})
```

**Same fix needed for:** `bitcoin-status` polling call (line ~135)
