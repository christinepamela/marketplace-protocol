## 📚 REFERENCE DOCUMENTS

### Quick Reference: Status Values

**Order statuses:**
```
draft → payment_pending → paid → confirmed → shipped → delivered → completed → cancelled
```

**Quote statuses:**
```
pending → accepted / rejected / expired
```

**Shipment statuses:**
```
pending_pickup → picked_up → in_transit → out_for_delivery
→ delivered / failed_delivery / returning / returned / lost / cancelled
```
